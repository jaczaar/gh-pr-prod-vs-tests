GitHub token (private repos)

Optional. Required only for private PRs. Use a fine-grained PAT with read access to Pull requests. Stored in chrome.storage.sync.
Create one: github.com/settings/personal-access-tokens/new

Test path patterns

One JavaScript regex per line. Matched against the file path. These are added to the built-in defaults, not replacing them.

Built-in defaults
  • \.test\.(t|j)sx?$
  • \.spec\.(t|j)sx?$
  • \.cy\.(t|j)sx?$
  • (^|/)__tests__/
  • (^|/)__mocks__/
  • (^|/)__snapshots__/
  • (^|/)tests?/
  • (^|/)e2e/
  • (^|/)cypress/
  • (^|/)playwright/
  • (^|/)fixtures?/
  • _test\.(py|go)$
  • _spec\.rb$
  • (^|/)test_[^/]+\.py$
